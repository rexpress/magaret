Requirement
===========
- Node 5.xx or later
- Git

Install and Run
===============

**Clone repository**
```{r, engine='bash'}
$ git clone https://github.com/rexpress/magaret.git
```

**Change Directory**
```{r, engine='bash'}
$ cd magaret
```

**Module Install**
```{r, engine='bash'}
$ npm install
```

**Run Application**
```{r, engine='bash'}
$ npm start
```

**Update to latest**
```{r, engine='bash'}
$ git pull
$ npm install
```
